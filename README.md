# 🚗 ESP8266 WiFi Race Car with Joystick & Buzzer

Control your DIY WiFi race car using a smooth mobile web joystick.
Includes real-time motor control and buzzer horn using WebSocket.

> Built for fun. Engineered with chaos. Powered by WiFi 😎

---

## 🔥 Features

- 📡 WiFi Controlled (No App Required)
- 🎮 Smooth Touch Joystick
- ⚡ Real-Time WebSocket Communication
- 🔊 Buzzer Button (Hold to Honk)
- 📱 Mobile Responsive UI
- 🌙 Dark Cyber Theme

---

## 🧠 How It Works

1. ESP8266 connects to WiFi
2. Hosts a web server
3. WebSocket sends real-time joystick values
4. Motor speed calculated dynamically
5. Buzzer controlled independently

---

## 🛠 Hardware Used

- NodeMCU ESP8266
- L298N Motor Driver
- 2 × DC Motors
- Buzzer
- Battery Pack
- Jumper Wires

---

## 🚀 Setup Instructions

1. Install ESP8266 board in Arduino IDE
2. Install required libraries:
   - ESP8266WiFi
   - ESP8266WebServer
   - WebSocketsServer
3. Update WiFi credentials in code
4. Upload to NodeMCU
5. Open Serial Monitor (115200)
6. Enter shown IP address in mobile browser
